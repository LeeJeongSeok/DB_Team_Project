# 배달 주문 시스템

## 배달 주문 시스템

### 배달 관리

- 배달 접수 번호
- 배달 접수 번호 조회
- 배달원 관리 

### 음식점 관리

- 음식점 등록
- 음식점 조회
- 음식점 수정
  - 메뉴 관리
    - 메뉴 등록
    - 메뉴 조회
    - 메뉴 수정
    - 메뉴 삭제

### 고객 관리

- 회원 등록
- 회원 조회
- 회원 수정
- 주문 내역
- 문의 코드
- 평점 및 리뷰 관리
- 결제 관리
  - 결제 등록
  - 결제 내역

### 주문 접수 관리

- 주문 받은 시간
- 주문 소요 시간
- 주문 취소
- 결제자 정보


### 문의 코드 (넘버로 코드 구분)

- 잘못된 조리 (1)
- 메뉴 누락 (2)
- 포장 불량 (3)
- 음식 전달 실수 (4)
- 기타 (5)

#### 기타

- 배달 업체 지정
- 주문 1건당 배달 1건
